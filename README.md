# simfonik

A curated archive of 90's rave DJ mixes.

**Live site**: [simfonik.com](https://simfonik.com)

## About

simfonik began in 2007 as an effort to preserve and share DJ mixtapes from the early 1990s Los Angeles rave scene. What started as a small personal collection has grown into a growing archive of DJ mixtapes from the early 90s era, documenting a period that helped define underground dance music culture.

## Features

- Growing archive of DJ mixtapes from the early 90s era
- Searchable by DJ name, mix title, or year
- DJ profiles with "also known as" aliases
- Audio streaming with playback coordination
- Responsive design optimized for all devices

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Hosting**: Vercel
- **Audio**: Dropbox CDN (for now)
- **Styling**: Tailwind CSS
- **Data**: JSON-based content management

## Project Structure

- `/web` - Next.js application
- `/specs` - Project specifications and documentation
- `/web/data` - Tape metadata and DJ aliases
- `/web/public/media` - Cover art and images
