# 0009 — SEO Improvements

## Scope
Improve SEO through semantic structure, metadata hygiene, image accessibility, and internal linking.

**No new pages, no URL changes, no new content blocks, no schema, no design changes.**

---

## Goals
- One clear primary topic per page
- Unique, deterministic metadata per page type
- Clean, consistent heading hierarchy
- Strong internal linking between related entities
- Descriptive image alt text without visual changes

---

## Global rules

### Heading hierarchy
- Exactly one `<h1>` per page in the DOM
- `<h1>` represents the page's primary entity or intent
- Global layout components must never render an H1
- Heading levels reflect document hierarchy, not visual styling
- Truncation is allowed in `<title>` values for display length only

### Metadata requirements
- Titles and meta descriptions must be:
  - Deterministic
  - Unique across all HTML URLs
  - Generated from page data, not generic templates
- Open Graph and Twitter metadata must mirror page intent
- Title truncation must not remove the primary entity

### Tape detail pages (`/tapes/[id]`)

**Structure:**
- H1: `tape.title`
- H2: Side titles in audio player section (`Side A`, `Side B`, or custom side titles)
- H3: Tracklist section headings (optional, when track data exists)
  - Format: `{{Side title}} – Tracklist`
- H3: `Comments`, `Leave a Comment`, `Archived Comments`

**Metadata:**
- Title: `{{DJ names (truncated)}} – {{Tape title}} ({{Year}})`
  - For 3+ DJs: Show first 2 DJs + `...`
  - Truncate by character limit (max ~60 chars) ensuring primary DJ always shown
  - Full format example: `DJ Trance, Doc Martin, ... – Essential Mix (1994)`
- Meta description: `{{Full DJ list}} – {{Tape title}} ({{Year}}){{source ? ` • Tape Source: ${source}` : ''}}`
  - No truncation in description
  - Include source attribution if present

**Images:**
- Cover image: `{{Tape title}} mixtape by {{Full DJ list}}`
- Side images: `{{Tape title}} – {{Side title}} image`
- Placeholder: `Blank cassette tape placeholder`

**Document hierarchy example:**
```
H1: Tape title
├── H2: Side A
├── H2: Side B
├── H3: Side A – Tracklist (conditional)
├── H3: Side B – Tracklist (conditional)
├── H3: Comments
├── H3: Leave a Comment
└── H3: Archived Comments
```

## Global layout fix

**Critical issue:**
The site header in `layout.tsx` currently renders an H1 for the site title:

```tsx

```

**Fix required:**
Change to `<div>` or `<span>` with identical styling. This H1 appears on every page, creating duplicate H1s.

```tsx
<div className="text-2xl font-bold">simfonik</div>
```

## Images

### Requirements
- All meaningful images must have descriptive alt text
- Decorative images must have empty alt attributes (`alt=""`)
- No changes to image sources, layout, or sizing

### Terminology
- Prefer "mixtape" over "cassette" for better search alignment
  - Users search "DJ mixtape", not "DJ cassette"
  - Example: `{{Title}} mixtape by {{DJ}}` not `{{Title}} cassette by {{DJ}}`

---

## Internal linking

### Anchor text rules
- Tape pages link to DJ pages using DJ names
- DJ pages link to tape pages using tape titles
- Index pages link to entity pages using entity names
- Avoid non-descriptive anchor text (no "click here", "read more", etc.)

### Card-based navigation
- Card-based navigation is acceptable when the card heading serves as the anchor text
- Invisible absolute-positioned links wrapping cards are fine if:
  - The visible card title is properly exposed to screen readers
  - The `aria-label` accurately describes the destination
  - Example: `aria-label="View {{tape.title}}"` with visible H2 containing tape title

## Implementation notes

### DJ name truncation logic
Current implementation shows first 2 DJs + `...` for 3+ DJs. Consider refining to:
- Truncate by character limit (max ~60 chars for titles)
- Always preserve at least the primary (first) DJ name
- Gracefully handle very long individual DJ names
- Ensure truncation doesn't cut mid-word

### Example truncation scenarios
```
1 DJ:  "DJ Trance – Essential Mix (1994)"
2 DJs: "DJ Trance, Doc Martin – Essential Mix (1994)"
3 DJs: "DJ Trance, Doc Martin, ... – Essential Mix (1994)"
5 DJs with long names: "DJ Trance, Doc Martin, ... – Essential Mix (1994)"
```
